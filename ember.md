# Ember.js

### Getting Started

- **[Ember.js](https://emberjs.com/)** - main site
- **Installation**
	- NPM
	- CLI

### Basic Example

Install Ember

	> npm install -g ember-cli@2.15

Create a New Application

	> ember new ember-quickstart

Change directory to _ember-quickstart_ and start the development server:

	> cd ember-quickstart
	> ember serve

Open Browser:

	http://localhost:4200
