# HTML5

The following document shows a simple progression for an HTML5 web page.

### doctype Element
	<!DOCTYPE html>

### html Element
	<!DOCTYPE html>
	<html>	
	</html>

### head Element
	<!DOCTYPE html>
	<html>
		<head>
			<title>Hello World!</title>
		</head>
	</html>

### body Element
	<!DOCTYPE html>
	<html>
		<head>
			<title>Hello World!</title>
		</head>
		<body>
		</body>
	</html>

### meta Elements
	<!DOCTYPE html>
	<html>
		<head>	
			<meta charset="UTF-8">
			<meta name="author" content="Doug Duril" />
			<meta name="description" content="Template Example" />
			<title>Hello World!</title>
		</head>
		<body>
		</body>
	</html>
