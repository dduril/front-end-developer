# Vue.js

### Getting Started

- **[Vue.js](https://vuejs.org/)** - main site
- **Installation**
	- Use a CDN
	- NPM
	- CLI

### Basic Example

Install Vue.js

	> npm install -g vue-cli

Create a directory and change into it:

	> vue init webpack my-project
	> cd my-project

Install dependencies and launch server:

	> npm install
	> npm run dev

Browser will open or open browser of choice:

	http://localhost:8080
